RÈGLES :

– Il s’agit d’un jeu de Shifumi, les symboles changes mais fonctionnent de la même façon : pierre (○) → ciseaux (×) → feuille (∆).

– Pour jouer les symboles, vous pouvez soit les déplacer, soit cliquer sur la touche correspondante : ○ (a)ou(←), × (b)ou(↓), ∆ (c)ou(→).

– Vous affrontez l’IA en 3 manches gagnantes.

– À tout moment, vous pouvez rouvrir à nouveau le menu pour vérifier les règles.
